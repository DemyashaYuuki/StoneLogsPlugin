# StoneLogs

Плагин для **Paper 1.21.8**, который делает все блоки брёвен заметно сложнее для добычи:

- все блоки из тега `#minecraft:logs` добываются по кастомной скорости;
- без топора ломаются примерно как камень без правильного инструмента;
- топоры заметно ускоряют добычу;
- учитываются **Efficiency**, **Haste**, **Mining Fatigue**, вода и штраф за добычу в воздухе;
- анимация ломания блока показывается игроку в процессе;
- логика применяется в режиме **Survival**.
