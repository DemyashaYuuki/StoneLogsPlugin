# StoneLogs

Плагин для **Paper 1.21.8**, который делает все блоки брёвен заметно сложнее для добычи:

- все блоки из тега `#minecraft:logs` добываются по кастомной скорости;
- без топора ломаются примерно как камень без правильного инструмента;
- топоры заметно ускоряют добычу;
- учитываются **Efficiency**, **Haste**, **Mining Fatigue**, вода и штраф за добычу в воздухе;
- анимация ломания блока показывается игроку в процессе;
- логика применяется в режиме **Survival**.

## Что входит

- Maven-проект
- `plugin.yml`
- GitHub Actions workflow для автоматической сборки `.jar`

## Сборка через GitHub

1. Загрузи содержимое этого архива в свой репозиторий GitHub.
2. Открой вкладку **Actions**.
3. Запусти workflow **Build StoneLogs** или просто запушь изменения.
4. Готовый `.jar` появится в **Artifacts**.

## Локальная сборка

```bash
mvn clean package
```

Готовый файл появится в:

```bash
target/stonelogs-1.0.0.jar
```

## Замечание

Плагин меняет именно **скорость добычи** брёвен, а не внутреннюю NMS-hardness самих блоков.
Это самый безопасный и переносимый вариант для Paper-плагина без привязки к внутренностям конкретной версии сервера.
